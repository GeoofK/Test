Heading

we use h1 - h6 elements to write headings on a website. H1 has most importance and H6 being the list important.

Paragraphs

To write a paragraph, we use the P element where all the content is written in between the P element.

images

To put an image in a website we use the img element which contains attributes inside it.
Below are some of the attributes;

  • src - gives the location of the image
  • alt - used as a failsafe inacase the image fails to display
  • widht, height - used to style the image
